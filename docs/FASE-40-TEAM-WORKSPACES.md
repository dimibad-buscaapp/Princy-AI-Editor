# FASE 40 — Team Workspaces

Organizações, times, membros e workspaces compartilhados.

## Endpoints

- `GET/POST /api/orgs`
- `GET/POST /api/teams`
- `GET /api/workspaces/shared`

## Models

`Organization`, `Team`, `TeamMember`, `Project.teamId`

## Access

`assertProjectAccess` estendido para membros do time.
